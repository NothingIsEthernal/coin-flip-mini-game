# 🪙 Coin Flip Mini Game

A simple and fun coin flip prediction game built with HTML, CSS, and JavaScript.

## 🎮 Game Features

- **Choose your prediction**: Select Heads or Tails
- **Flip the coin**: Click the coin or buttons to flip
- **Win/Lose tracking**: Keep track of your wins and losses
- **Smooth animations**: Beautiful 3D coin flip animation
- **Responsive design**: Works on desktop and mobile devices
- **Reset stats**: Clear your stats anytime

## 🚀 How to Play

1. Click **Heads** or **Tails** button to make your prediction
2. The coin will flip automatically
3. If you predicted correctly, you win! 🎉
4. If not, you lose 😞
5. Check your stats and try again!
